# 项目简介

## 核心功能
- 文章推荐：首页文章推荐采用协同过滤算法
- 前后端代码：提供完整的代码，包括sql文件
- 文档支持：赠送万字文档，提供详细的使用说明
- 视频教程：提供参考导入调试视频、讲解视频以及开发工具安装包
- 个人中心：用户可以管理个人信息、密码、聊天记录、发布内容、收藏文章
- 管理员功能：包括Home、我的信息、博主管理、普通管理员管理、博客文章管理、文章类型管理、系统公告管理、交流社区、系统管理等

## 适用场景
- 博客网站开发
- 文章推荐系统搭建
- 教育培训资料

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
